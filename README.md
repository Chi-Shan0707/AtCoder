# AtCoder Heuristic Contest

AtCoder Heuristic Contest (AHC) solutions. AHC is the only category I compete in at AtCoder -- it's the best kind of problem for keeping the brain sharp: open-ended optimization, no single correct answer, always room for improvement.

## Why AHC

Unlike typical competitive programming where the answer is binary (AC/WA), heuristic contests reward continuous improvement. Every tweak to your heuristic, every clever pruning, every local-search trick earns incremental score. It's a different kind of thinking -- more engineering than math.

## Layout

Each contest lives in its own directory:

```
AHCxxx/
├── README.md          # Problem summary, approach notes, score history
├── *.cpp              # Solution source
├── tools/             # Verifiers, visualizers, input generators
└── test/              # Test I/O files (git-ignored)
```
